Memoize the ``MappedTaskUpstreamDep`` upstream task-instance query per scheduling cycle to remove a per-mapped-index N+1. The dependency previously re-ran an identical ``SELECT`` over the mapped task's expansion-source task instances once for every mapped index (e.g. ~1000 identical queries at width 1000); the result is now cached on the per-cycle ``DepContext`` and reused across all mapped indexes of the same task.
